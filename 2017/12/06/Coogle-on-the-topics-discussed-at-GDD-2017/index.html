Coogle on the topics discussed at GDD India 2017

If you are not aware of GDD India 2017, then you must read my
previous blog post

Coogle is a web app which allows us to mind map concepts.
Its quite colorful and fun way to show the topics covered at GDD India 2017.

The mind map is on
Coogle

Open the link and scroll around to view the topics.

At a high level, the main topics were

  • Web
    • Progressive web apps & AMP
  • Android
    • Kotlin
    • Android things
  • Machine learning
    • Tensor flow
    • Tensor flow lite
  • Firebase
    • Firebase cloud
  • Others
    • Google assistant
    • Google dialogflow

Coogle,
